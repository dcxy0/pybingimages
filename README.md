## pybingimages - Python网络抓取工具及其它一些工具
###关于Python版本
    请使用Python3.3以上版本[之前有一个面向Python2.7的版本,不过已经被升级了..].
###依赖
    至少需要Urllib3,Pillow[在Python2和Python3,PIL共存时,会出现问题]
###为啥会删掉一个函数
	在Code中删掉了一个获取图片URL的函数[在bingimages.py文件中].可以自行实现,或Email我.
